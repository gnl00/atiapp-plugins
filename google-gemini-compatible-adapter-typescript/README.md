# Google Gemini Compatible Adapter (TypeScript)

TypeScript version of the local Google Gemini request-adapter plugin.

## Build

```bash
pnpm install
pnpm build
```

The built entry is:

- `dist/main.js`

## Import

Import this directory from Settings -> Plugins.

## Notes

- API base URL example: `https://generativelanguage.googleapis.com/v1beta`
- Adapter provider type: `gemini`
- Adapter api version: `v1beta`
